# Run the API

--- 

## _Package installation_

#### Dev Dependencies
> Node Js dependency: `npm i -D nodemon` 

> Express Js Installation: `npm i express`


## Start the server
> `npm run dev`

## Test the API

 _Server_: `localhost:3000`

 _Base route:_ `/api/`

 _Full base route:_ `localhost:3000/api/`
